# 📖 Mini Guia de Estudos

## O que é Computação em Nuvem?

Computação em nuvem é o fornecimento de recursos computacionais pela internet, permitindo escalabilidade, disponibilidade e redução de custos.

---

## Modelos de Nuvem

- Pública
- Privada
- Híbrida

---

## Modelos de Serviço

- IaaS
- PaaS
- SaaS

---

## Bancos de Dados Relacionais

Características:

- tabelas
- chaves
- SQL
- consistência

Serviço Azure:

- Azure SQL Database

---

## Bancos Não Relacionais

Características:

- documentos
- chave-valor
- grafos
- colunas

Serviço Azure:

- Cosmos DB

---

## Armazenamento

Blob Storage

Arquivos grandes.

Data Lake Storage

Grandes volumes de dados para Analytics.

Queue Storage

Mensageria.

Table Storage

NoSQL simples.

---

## Processamento Analítico

Data Warehouse

Dados estruturados para BI.

Data Lake

Dados estruturados e não estruturados.

Lakehouse

Combina Data Warehouse com Data Lake.

---

## Ferramentas Analíticas

- Azure Synapse Analytics
- Microsoft Fabric
- Power BI

# 📚 Glossário

Azure SQL Database → Banco de dados relacional.

Cosmos DB → Banco NoSQL.

Blob Storage → Armazenamento de arquivos.

Data Lake → Repositório de dados brutos.

Data Warehouse → Base analítica.

Lakehouse → União entre Data Lake e Warehouse.

ETL → Extract, Transform, Load.

ELT → Extract, Load, Transform.

OLTP → Processamento transacional.

OLAP → Processamento analítico.
